Map your AMML Staff IDs to ZKTeco User IDs (numeric). This is used to correctly match imported attendance records. ZK User IDs are set during fingerprint enrollment on the device.
AMML IDNameMarketZK User ID
Step 1 — Connect MB10 to LAN
Connect the device to the same network as this computer via Ethernet. Default IP: 192.168.1.201. Set via: Menu → Comm → Ethernet.
Step 2 — Enroll Staff Fingerprints/Faces
On the device: Menu → User Mgt → New User. Assign a numeric User ID (e.g. 1, 2, 3…) and enroll fingerprint or face. Record the ZK User ID in the Staff Enroll tab above so AMML can match them.
Step 3 — Export Attendance Logs
Option A (USB): Insert USB drive → Menu → USB Manager → Download AttLog. A file named attlog.dat will be saved.
Option B (Software): Install ZKAccess or ZKTime on a PC, connect to device IP, export as CSV.
Step 4 — Import into AMML
Go to Import Logs tab → select the exported file → choose the market → click Import to AMML.
💡 Advanced: HTTP Push Protocol
For automatic real-time sync, configure the MB10 to push data:
Menu → Comm → Cloud Server → Server Address → set to your server IP + port. Requires a backend receiver (Node.js / Python) to accept POST requests and forward records into AMML.

Fingerprint Capacity: 500 · Face Capacity: 200 (expandable to 300)
Users: 1,000 · Attendance Records: 50,000
Comms: TCP/IP · USB · RS232/485
Verification: Fingerprint · Face · Password · Card (optional)

The Butech device identifies staff by name (no numeric ID). AMML matches imported names to registered staff automatically. Review and correct any mismatches here before importing.

Step 1 — Register Staff on Device
An admin enrolls each staff member's fingerprint via the device menu. The name entered during enrollment must closely match the name in AMML for automatic matching to work.
Step 2 — Daily Clock In / Out
Staff place their finger on the terminal at resumption and at close of work. The device records both times automatically.
Step 3 — Download Report via USB
Insert a USB flash drive into the Butech terminal. The device writes a locked Excel report instantly — no software required. The file is tamper-evident (write-protected) and opens directly in Microsoft Excel or Google Sheets.
Step 4 — Import into AMML
Go to Import Excel Report tab, upload the file, confirm the market, and click Import to AMML. Records are matched by staff name and merged into your attendance table.
Butech FP Terminal — Report Columns
The USB Excel export contains: Staff Name · Department · Clock-In Time · Clock-Out Time · Hours Required · Hours Worked

Key Features: High-accuracy fingerprint · Rapid identification · Durable · No battery · Eco-friendly · Direct-to-light sensor · Locked tamper-proof Excel export
